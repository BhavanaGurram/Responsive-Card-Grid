# Responsive-card-grid-with-hover-animations
Responsive Grid Layout:
    For large screens (min-width: 1024px):
    The grid should have 2 rows and 3 columns.
    For medium screens (min-width: 768px and max-width: 1023px):
    The grid should have 3 rows and 2 columns.
    For small screens (max-width: 767px):
    The grid should have a single column with all cards stacked vertically.
    There should be a gap between the cards in the grid.
Card Hover Animations:
    When a user hovers over a card:
    The card should scale up slightly.
    The card should display a semi-transparent black overlay.
    The text color inside the card should change to white.
    The button inside the card should change its background color to white and text color to black.
